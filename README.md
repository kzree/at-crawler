# AT-crawler

Webcrawler for the Arvutitark webstore.

## Requirements

- Python 3.x
	- Requests
	- bs4

## Usage

``python3 crawler.py <item_name>``

### Optional parameters

- -min <number>: Minimum price for crawled items
- -max <number>: Maximum price for crawled items
